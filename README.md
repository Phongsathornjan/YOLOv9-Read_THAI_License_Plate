How to install my project.<br>
follow as this command.<br>
  1.) python -m venv myenv.<br>
  2.) myenv\Script\activate.<br>
  3.) pip install -r requirement.txt.<br>
  4.) python app.py.<br>

Server will be run on localhost:5000.<br>
